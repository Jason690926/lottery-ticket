# 威力彩選號研究 — 結案報告

> 分析期間：2026-05-03 ～ 2026-05-09  
> 資料範圍：2008-01-24 ～ 2026-04-30（共 1,907 期）  
> 訓練集：1,807 期｜Holdout：最後 100 期（**全程未解封**）

---

## 一、研究目標

檢驗威力彩歷史開獎資料是否存在可統計量化的規律，足以輔助選號。

核心假說（D-revised）：上期出現的號碼組合，能影響下期號碼的出現機率。

---

## 二、資料與方法

### 資料來源
- 政府開放資料平台 D423F（台彩官方 CDN）
- 一區：1–38 選 6；二區：1–8 選 1
- 訓練 / Holdout 切分於資料下載時鎖定，Holdout 全程未碰

### 統計框架
- **H₀**：每期開獎為 i.i.d.（獨立同分布）隨機事件
- **多重比較校正**：FDR Benjamini-Hochberg（q < 0.05）
- **結案門檻**：所有探索方向均無通過 FDR 校正的顯著訊號

### 3.3 Order C（Pair → 下期單號）

- 通過 ≥20 次共現門檻：**703 對**（所有 C(38,2) = 703 對均通過）
- 總檢定數：703 × 38 = **26,714 個**
- FDR q < 0.05：**0 個**

⚠️ 功效說明：中位樣本 n = 39，對 lift = 1.5 的統計力僅 ≈ 30%。  
「無顯著」不等於「效果為零」，但結合 lag 分析的高功效結果，整體方向一致。

→ 雙號條件預測方向**無訊號**。

## 五、結論

**威力彩歷史開獎資料（2008–2026，1,807 期訓練集）在以下所有統計維度上，均與理論 i.i.d. 隨機過程無法區分：**

1. 靜態號碼頻率
2. 跨期 Lag-1/2/3 Transition（單號 → 單號）
3. 同期 Pair → 下期單號
4. 長期趨勢、時段切割、近期窗口
5. 時間序列自相關（ACF/Ljung-Box）、週期性（Fisher's g）、日曆效應（開獎日）

依照研究開始時設定的**證偽門檻**（任何方向 FDR q < 0.05 才有訊號），本研究結果為**負結果**，無法支持以歷史資料預測下期號碼的假說。

### 誠實說明

- **Holdout 100 期從未使用**：所有分析皆在訓練集進行，未有資料洩漏。
- **低功效方向**：Order C（n≈39）的功效不足，無法偵測小效果（lift < 1.5）。若要提高功效，需要更長的歷史資料（目前 18 年已是極限）。
- **探索範圍外**：深度學習序列模型（LSTM/Transformer）未測試，但鑑於統計顯著性全面為零，預期結果相同。

### 這個結果合理嗎？

是的。威力彩採用：
- 美國 Smartplay Inc. 開獎機（WLA 認可、TÜV 認證）
- 每月維護 + 每年原廠檢驗
- 安永會計師事務所公正監督

三方嚴格把關的系統設計目標正是 i.i.d.，找不到訊號是符合預期的結果。
